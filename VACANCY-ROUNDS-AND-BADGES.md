# Vacancy rounds and menu badges

This release adds:

- Vacancy rounds for returned jobs.
- Historical applications from previous rounds.
- The same freelancer may apply again after a vacancy returns.
- Previous assignee information is retained on the publication.
- Cleaner Current RMS status wording on Applications.
- Red menu badges with white numbers:
  - Available Work for freelancers.
  - Applications for approvers.
- Badges hide at zero and refresh every 60 seconds.
- Slightly denser cards and metric tiles.

Install with:

npm run migrate
